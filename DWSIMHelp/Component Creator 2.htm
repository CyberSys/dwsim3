UNIFAC Group Definition

This tab is used to define UNIFAC-groups of the component. UNIFAC groups again usually can be divided into even smaller Joback subgoups.



  1. Subgoup
    Name of subgoup
     
  2. Amount
    Numer of subgoups in molecule
     
  3. Example
    Example component to show how this subgoup looks like.
     
  4. Tooll tip
    Each example component will display a tool tip window which gives you all available information of this subgoup and its example component.
     
  5. DortmundDatabase
    This link will guide you to the DartmundDatabase where you can get help to find the correct assignment of UNIFAC groups to your component.
