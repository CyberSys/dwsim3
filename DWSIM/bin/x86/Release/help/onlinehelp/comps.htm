Componentes / Substâncias

A primeira coisa que se deve fazer ao criar uma nova simulação é adicionar as substâncias que estarão presentes. Internamente, as substâncias do DWSIM são na verdade grupos de dados e parâmetros físico-químicos (ex. propriedades críticas, fator acêntrico, curva de pressão de vapor, curva de viscosidade) que representam a substância sendo adicionada. Estes dados são necessários para que os modelos termodinâmicos possam calcular a distribuição dos componentes nas fases suportadas pelo DWSIM (líquido e vapor) assim como as propriedades destas fases, que são funções das quantidades relativas de cada componente.

Para adicionar componentes (substâncias) à simulação, utilize a aba Componentes da janela de configuração da simulação.

Você pode procurar o componente pelo seu nome (caixa de texto Procurar) ou selecionar o componente diretamente da lista no lado esquerdo. Para adicioná-lo a simulação, basta

  • dar dois cliques em cima da linha da tabela correspondente ao componente, ou
  • com o componente selecionado, clicar em "Adicionar".

Para remover o componente da simulação, efetue o processo inverso. Com o componente selecionado na lista à direita, dê dois cliques com o mouse em cima do nome do mesmo ou clique no botão "Remover".

Para remover todos os componentes da simulação, clique em "Limpar Lista".

No DWSIM também é possível especificar o seu próprio componente caso ele não esteja presente no banco de dados. Existem três diferentes formas de fazer isso, mas a mais recomendada é utilizar o Criador de Componentes.
