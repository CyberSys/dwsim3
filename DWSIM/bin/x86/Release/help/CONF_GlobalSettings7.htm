Global Settings

This tool is used to edit global settings of DWSIM

Other Configurations

Messages


Show tips: shows context-sensitive usage tips on the information window.
Check for updates on startup: DWSIM checks Sourceforge for available updates.
Show 'what's new' window: display a window with information about what's new on the current DWSIM build when opening a flowsheet.

Debug mode


Debug level: Select how much internal information is displayed during calculations. Select "none" for maximum speed.
Redirect console output: Check this to show debug information on an internal DWSIM console window.

UI Language


Selects the User Interface language for DWSIM.

Compatibility


Enables or disables loading of binary files (".dwsim" extension) using legacy settings. Use this to load old simulation files into DWSIM 3+ versions.
