Global Settings

This tool is used to edit global settings of DWSIM

Component Databases

Component Databases

This table lists all component databases beeing used in DWSIM. During startup of DWSIM all components of these databases are loaded.

Symbols


Lock: This is a system database which is not allowed to be removed.

Cross: This database is not allowed to be edited.

Disconnect: This will remove the database from list.

Edit: Opens Database Manager to edit this database.

Tasks


Add User Database: You can add a user database with your own components. User databases contain user defined components which were created by the Component creator utility.

Add ChemSep(TM) Database: You can add a ChemSep compound database manually as well. The ChemSep database in XML format is named "chemsep1.xml", and it is normally located inside the "pcd" folder where ChemSep is installed.

Replace components option: If this checkbox is checked, user components will replace existing components if the component ID is identical.
