Reactors

The reactors in DWSIM are specialized modules that solve a particular set of reactions in sequence or in parallel.

There are four different types of reactors available:
Type Description
CSTR: Continuous Stirred Tank Reactor
PFR: Plug Flow Reactor
Gibbs: Gibbs Minimsation Reactor
Equilibrium: Equilibrium Reactor
Conversion: Conversion Reactor

To run a simulation of a reactor it is necessary to define the chemical reactions which are to be used in this specific reactor. This is done in simulation thermodynamic setup with the "Reaction Manager" (see separate documentation).
The principle is to either define reactions of "Equilibrium", "Conversion" or "Kinetic" type. One or more reactions are combined to define a "Reation Set" which is used for calculation of the reactor.

Input Parameters

All reactors share the same basic interface. There must be set the
  • input and output streams
  • energy streams that represent the heat exchanged with the environment
  • reaction set to be used
  • mode of operation (isothermal or adiabatic)
  • pressure drop through the reactor.

For the PFR and CSTR, it is also necessary to define
  • volume of the reaction medium

For the Gibbs Reactor you select the calculation mode:
  • Diect Minimization: Define components taking part in reactions and the element matrix.
  • Reaction Extend: Select reaction set which defines extend of each reaction.


Output Parameters

As results the conversions of the components involved in the reactions, the variation in temperature and the heat exchanged in the reactor are shown.

For the PFR the profiles of concentration of reactants and products (see figure) is shown along the longitudinal axis of the reactor (assuming the concentration does not vary radially).
