Component Creator

Introduction

This tool is used to define new components which are not in any standard databases of DWSIM available.
New compents are saved into a user database. For creation of a simulation in DWSIM you can load these user defined components into your simulation. Storing user components in a database allows to use them in all simulations.

General procedure for creating a new component:

  1. Defining Name, CAS-Number and Database-ID
  2. Defining structure of molecule
  3. All necessary properties are estimated from molecular structur by DWSIM
  4. Replace estimated parameters by experimental values where available (Literature or own data)
  5. Save definition file
  6. Save component to user database
